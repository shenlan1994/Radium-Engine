void Ra::Core::AdjacencyQueries::getVertexFirstRing ( const TriangleMesh &  mesh,
const HalfEdgeData &  heData,
VertexIdx  vertex,
std::vector< VertexIdx > &  ringOut 
)

Gets the neighbors of a vertex in clockwise order.

  • If the vertex has a regular neighborhood (i.e. it doesn't sit on the border of a mesh), then ringOut forms an actual ring and each vertex in it is adjacent to the previous and next in the array, including the first and last.
  • If the vertex is located on the edge but all neighbors still form a connected graph (i.e. there's only one "missing link" in the ring) then ringOut will be in the correct order, each vertex adjacent to the previous and next in the array, except the first and last.
  • If there are multiple missing links, this function will assert. Consider using getVertexNeighbors instead.
